/*

[DESCRIPTION]

- Check for invalid OID value in context of object dependencies

[SOURCE]

- SAP Note 1969700

[DETAILS AND RESTRICTIONS]

- Can only be executed with SAP_INTERNAL_HANA_SUPPORT role or as SYSTEM

[VALID FOR]

- BASE_OID:        ID of base object
- BASE_TYPE:       Type of base object
- DEPENDENT_DBID:  Database ID of dependent object
- DEPENDENT_OID:   ID of dependent object

- DEPENDENCY_TYPE: Dependency type
- COMMAND:         Cleanup command (execute with care!)

*/

  BASEOID BASE_OID,
  BASETYPE BASE_TYPE,
  DEPENDENTDBID DEPENDENT_DBID,
  DEPENDENTOID DEPENDENT_OID,
  DEPENDENTTYPE DEPENDENT_TYPE,
  DEPENDENCYTYPE DEPENDENCY_TYPE,
  'CALL SYS.CHECK_CATALOG_DEV(' || CHAR(39) || 'DELETE_DEPENDENCY' || CHAR(39) || ',' || CHAR(32) || CHAR(39) || CHAR(39) || ',' || CHAR(32) || CHAR(39) || CHAR(39) ||

  DEPENDENTOID || CHAR(32) || 'DEPDENDENTTYPE:' || DEPENDENTTYPE || CHAR(32) || 'TYPE:' || DEPENDENCYTYPE || CHAR(39) || ')' || CHAR(59) COMMAND
FROM
  SYS.P_OBJECTDEPENDENCY_
WHERE
  BASEOID = 0 OR DEPENDENTOID = 0
